Play a random sound from the Hearthstone soundpack to test the audio.

## Instructions

1. Read `meta.json` from the repository root to get all available sounds
2. Randomly select one sound file path
3. Play it using `afplay` command
4. Report which sound was played

Use the Bash tool to read the metadata and play the sound.
